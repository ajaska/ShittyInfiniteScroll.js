ShittyInfiniteScroll.js

If you've always wanted to add infinite scrolling to your website, but don't know how or can't find the time to do so, ShittyInfiniteScroll.js is here to help.

This drop-in solution will add infinite scrolling to your page -- for free, with no coding knowledge required.

Benefits include:

  1. Make the page feel laggy by inserting tons of DOM elements!
  2. Give users who dare reach for the footer a hard time!
  3. I will revise ths copy later

To add this to your site, simply add this script tag to your page: 

<script src="http://www.ajaska.com/ShittyInfiniteScroll.js/ShittyInfiniteScroll.js"></script>

Or, if you want to host it yourself, you may find the latest release via the Github repo.
